🎯 Git Quest 

Git Quest Pro é uma plataforma interativa para aprender Git e ganhar XP completando desafios!
O projeto guia você por níveis com missões práticas, exibindo progresso, streaks e aprendizado focado no controle de versão.

🌐 Acesse online:
➡️ https://janainamorais-dev.github.io/GIT_PRATIC/

🚀 Sobre o Projeto

O objetivo deste projeto é ajudar iniciantes a dominar comandos e conceitos básicos de Git e GitHub, através de um ambiente gamificado.
Você avança pelas missões, acumula XP e registra seu progresso ao completar tarefas de Git diretamente no navegador.

📌 Funcionalidades

Este projeto implementa:

🎮 Mapa de missões progressivo
🧠 Objetivos por nível
💡 Dicas e instruções contextualizadas
🔁 Contador de streaks (sequências de acertos)
💻 Terminal interativo em Bash
📈 Exibição de status: XP, comandos executados e progresso
🧭 Como Usar
Acesse a página hospedada.
Navegue pelos níveis disponíveis.
Leia o objetivo e as instruções de cada missão.
Execute comandos Git no terminal embutido.
Complete a tarefa para ganhar XP e avançar.
📦 Tecnologias Utilizadas

Este projeto é construído com:

🧾 HTML5 — Estrutura da interface
🎨 CSS3 — Estilização e visual responsivo
⚙️ JavaScript — Lógica interativa e validação
